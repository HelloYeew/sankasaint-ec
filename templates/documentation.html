REST API (Beta)

The API has not finished yet, but you can already use it. The API is still in development and may change.

Sankasaint EC system has a full REST API that allows you to access all the data in the system. The API is available at /api/ and is available to all users. The API is available in JSON format only.

The API has powered by Django REST Framework with the power of drf-yasg for documentation.

Our documentation have 2 versions, Swagger and ReDoc so you can choose which one you like.
